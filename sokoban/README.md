Maven is needed.

Install PDDL4J with the script `installMVNpddl.sh`.

You can use the script `launch.sh` to start the program.

To change levels you will need to modify `SokobanMain.java`.

See planning solutions at http://localhost:4200/test.html or modify localhost with the VM IP address.

You can use the script `testAgent.sh` to start only the agent.
